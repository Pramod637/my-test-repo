# Git Demo

Ye file Git learning demo ke liye banayi gayi hai.

## Basic Git Commands

1. `git init`
   - Naye repository ko initialize karne ke liye.
2. `git add .`
   - Saare changes ko staging area mein daalne ke liye.
3. `git commit -m "Initial commit"`
   - Staged changes ko commit banane ke liye.
4. `git status`
   - Repository ka current status dekhne ke liye.
5. `git log`
   - Commit history dekhne ke liye.

## Example Workflow

```bash
git init
git add .
git commit -m "First Git demo commit"
git status
git log
```

## Tip

- Git seekhne ke liye practice jaruri hai.
- Har commit mein clear message likho.
- Branching aur merging bhi dheere-dheere samjho.
