💡 HOW IT WORKS
===============

STEP 1: User drags over text
  └─> Browser fires 'mouseup' event

STEP 2: Content script detects mouseup
  └─> Reads selected text via window.getSelection()

STEP 3: Text validation
  └─> Checks if at least 2 characters selected
  └─> Checks if extension is enabled
  └─> Checks if selection is not in input field

STEP 4: Copy to clipboard
  └─> Tries modern Clipboard API
  └─> Falls back to document.execCommand('copy')

STEP 5: Show notification
  └─> Floating "✓ Copied!" toast appears at cursor
  └─> Fades out after 1.5 seconds


🎯 SPECIAL YOUTUBE HANDLING
===========================

YouTube is a complex Single-Page App (SPA) that can intercept events.
The extension handles this by:

1. Using capture phase listeners (third parameter = true)
2. Adding a small delay (100ms) after mouseup on YouTube
3. Re-listening on body element (in addition to document)
4. Supporting video titles, descriptions, comments, transcripts, chapters
5. Handling YouTube's yt-navigate-finish event for page transitions

🆘 TROUBLESHOOTING
==================

Q: Toast notification not showing?
A: Make sure you're selecting at least 2 characters
   Check that extension is enabled (green toggle)
   Check F12 Console for errors

Q: Copy not working on some sites?
A: Some sites block the Clipboard API
   Extension falls back to execCommand automatically
   Try refreshing the page

Q: Extension disappeared?
A: It might have been disabled
   Go to chrome://extensions/ and enable it
   Or load it again

Q: Want to see debug logs?
A: Open F12 → Console tab
   Look for messages starting with [DragToCopy]

Q: Icons not showing?
A: They were auto-generated ✓
   Check: c:\Users\pc\chrome extention\icons\
   All three PNG files should be there
